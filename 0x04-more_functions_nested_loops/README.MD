# C - More Functions, More Nested Loops

## Description

This project focuses on practicing more functions and nested loops in C.
The tasks involve writing functions that work with characters, numbers, and patterns using loops.
All output is done using the `_putchar` function as required.

## Learning Objectives

* Understand how to create and use functions in C
* Practice nested loops
* Work with ASCII characters
* Print patterns using loops
* Follow Betty coding style guidelines

## Requirements

* Allowed editors: `vi`, `vim`, `emacs`
* Compiled on **Ubuntu 20.04 LTS**
* Compilation flags:

```
gcc -Wall -Werror -Wextra -pedantic -std=gnu89
```

* All files must end with a new line
* No global variables allowed
* Maximum of **5 functions per file**
* Only `_putchar` can be used for output (except where allowed)
* All prototypes must be declared in `main.h`
* Code must follow **Betty style**

## Files

| File                   | Description                                      |
| ---------------------- | ------------------------------------------------ |
| 0-isupper.c            | Function that checks for uppercase characters    |
| 1-isdigit.c            | Function that checks if a character is a digit   |

| 3-print_numbers.c      | Prints numbers from 0 to 9                       |
| 4-print_most_numbers.c | Prints numbers from 0 to 9 except 2 and 4        |
| 5-more_numbers.c       | Prints numbers from 0 to 14 ten times            |
| 6-print_line.c         | Draws a straight line                            |

| 8-print_square.c       | Prints a square using `#`                        |
| 9-fizz_buzz.c          | Prints numbers from 1 to 100 with FizzBuzz rules |
| 10-print_triangle.c    | Prints a triangle using `#`                      |
| main.h                 | Header file containing function prototypes       |
