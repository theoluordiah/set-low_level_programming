# 0x04. C - More Functions, More Nested Loops

## Learning Objectives
- Understand nested loops and their applications
- Use functions to create reusable code
- Handle character classification (uppercase, digit)
- Print patterns using loops and conditionals
- Solve classic problems like Fizz-Buzz

## Project Structure
- `main.h` - Header with all function prototypes
- `0-isupper.c` - Check for uppercase character
- `1-isdigit.c` - Check for digit (0-9)
- `2-mul.c` - Multiply two integers
- `3-print_numbers.c` - Print 0-9
- `4-print_most_numbers.c` - Print 0-9 except 2 and 4
- `5-more_numbers.c` - Print 0-14 ten times
- `6-print_line.c` - Draw straight line
- `7-print_diagonal.c` - Draw diagonal line
- `8-print_square.c` - Print a square
- `9-fizz_buzz.c` - Fizz-Buzz test program
- `10-print_triangle.c` - Print a triangle

## Environment
- Ubuntu 20.04 LTS
- gcc with flags: -Wall -Werror -Wextra -pedantic -std=gnu89
- Betty style checker
