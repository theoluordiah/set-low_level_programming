# 0x06. C - More Pointers, Arrays and Strings

## Learning Objectives
- Understand pointers and arrays at a deeper level
- Manipulate strings using pointer arithmetic
- Implement standard string functions: strcat, strncat, strncpy, strcmp
- Work with array reversal and case conversion
- Encode strings with custom mappings (leet)

## Project Structure
- `main.h` – header with all function prototypes
- `0-strcat.c` – concatenate two strings
- `1-strncat.c` – concatenate with a byte limit
- `2-strncpy.c` – copy a string with limit
- `3-strcmp.c` – compare two strings
- `4-rev_array.c` – reverse an array of integers
- `5-string_toupper.c` – convert lowercase to uppercase
- `6-cap_string.c` – capitalize words in a string
- `7-leet.c` – encode string into 1337

## Environment
- Ubuntu 20.04 LTS
- gcc with flags: -Wall -Werror -Wextra -pedantic -std=gnu89
- Betty style checker
