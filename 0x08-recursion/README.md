# 0x08. C - Recursion

## Learning Objectives
- Understand the concept of recursion
- Implement recursive functions without loops
- Master base cases and recursive cases
- Solve classic problems: string printing, factorial, power, square root, prime numbers

## Tasks
0. `_puts_recursion` – print a string using recursion
1. `_print_rev_recursion` – print a string in reverse
2. `_strlen_recursion` – calculate string length recursively
3. `factorial` – compute factorial recursively
4. `_pow_recursion` – compute power recursively
5. `_sqrt_recursion` – find natural square root recursively
6. `is_prime_number` – check if a number is prime recursively

## Environment
- Ubuntu 20.04 LTS
- gcc flags: -Wall -Werror -Wextra -pedantic -std=gnu89
- Betty style
