# 0x07. C - Even More Pointers, Arrays and Strings

## Learning Objectives
- Understand pointers to pointers and multidimensional arrays
- Implement common C library functions manually: memset, memcpy, strchr, strspn, strpbrk, strstr
- Work with 2D arrays and matrix diagonal sums
- Print a chessboard using 2D arrays

## Environment
- Ubuntu 20.04 LTS
- gcc with flags: -Wall -Werror -Wextra -pedantic -std=gnu89
- Betty style checker
