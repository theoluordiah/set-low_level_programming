# C - Pointers, Arrays and Strings

This project is part of the Low-Level Programming curriculum at SET, Elmwood Institute. It focuses on deepening understanding of pointers, arrays, and strings in C.

## Learning Objectives
- Understand pointers and how to use them
- Manipulate arrays and strings
- Differentiate between pointers and arrays
- Work with strings using C standard library functions (reimplemented)
- Understand variable scope

## Tasks
0. **98 Battery st.** - Write a function that takes a pointer to an int and updates the value to 98.
1. **Swap** - Swap the values of two integers.
2. **String length** - Return the length of a string.
3. **Print string** - Print a string followed by a new line.
4. **Print reverse** - Print a string in reverse.
5. **Reverse string** - Reverse a string in place.
6. **Print every other character** - Print every other character of a string.
7. **Print half of a string** - Print the second half of a string.
8. **Print array** - Print n elements of an array of integers.
9. **strcpy** - Copy a string.
